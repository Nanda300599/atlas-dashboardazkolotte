# AZKO LOTTE MALL - Store Performance Dashboard

Enterprise-grade frontend prototype built with HTML5, CSS3, JavaScript, Bootstrap 5, and Chart.js.

Structure:
- assets/css
- assets/js
- assets/components
- assets/images
- assets/icons
- index.html
- login.html
- dashboard.html
- settings.html

Quick start:
Open `login.html` in your browser (or run a static server).

Default demo credentials:
- username: `admin`
- password: `Admin@123`

Notes:
- Authentication is implemented using Local Storage for the initial prototype.
- Designed for easy migration to Firebase or server-side auth.
